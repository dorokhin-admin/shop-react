# React Shop 🛒

Интернет-магазин на React с корзиной, API и управлением заказами.

Функционал
-  Просмотр товаров
-  Добавление в корзину
-  Удаление товаров из корзины
-  Обновление количества
-  Работа с API (json-server)
-  Регистрация и авторизация 

Технологии
- React
- JavaScript 
- Zustand 
- React Router
- Fetch
- JSON Server
- CSS 

Структура проекта
src/
 api/  запросы к серверу
 components/  UI компоненты
 pages/  страницы
 store/  состояние 
 App.jsx

Установка и запуск
1. Клонировать проект
git clone https://github.com/dorokhin-admin/shop-react.git
2. Установить зависимости
npm install
3. Запустить проект
npm run dev
4. Запустить сервер
npm run server
